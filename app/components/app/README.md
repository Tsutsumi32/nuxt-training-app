# App Components

アプリケーション全体で使用されるコンポーネントを配置するディレクトリです。

## 配置するコンポーネント

- **レイアウト関連**: Header, Footer, Navigationなど
- **アプリケーション固有のコンポーネント**
- **サイト全体で共有されるコンポーネント**

## 命名規則

- ファイル名: PascalCase（例: `Header.vue`, `Footer.vue`）
- 使用時: `<AppHeader />`, `<AppFooter />`（Nuxtの自動インポートにより、`app/`ディレクトリ内のコンポーネントには`App`プレフィックスが付きます）

## コンポーネント一覧

- `Header.vue` - サイトヘッダーコンポーネント（ナビゲーションメニュー含む）
- `Footer.vue` - サイトフッターコンポーネント
